사용 방법
1
유튜브 영상 URL을 위에 붙여넣으세요. 예: youtube.com/watch?v=abc123 또는 영상 ID만 입력해도 돼요.
2
자막 언어와 PDF 스타일을 선택하세요. 쉐도잉 연습에는 쉐도잉용 스타일을 추천해요.
3
자막 가져오기를 누르면 자막이 표시되고, PDF 저장으로 다운로드할 수 있어요.
브라우저 제한 안내
!
브라우저 보안 정책(CORS)으로 인해 유튜브 자막을 직접 가져올 수 없는 경우가 있어요. 그럴 땐 아래 방법으로 자막 텍스트를 직접 붙여넣기 할 수 있어요.
직접 텍스트 붙여넣기
A
유튜브 영상 하단 …(더보기) → 자막 열기를 클릭하면 전체 자막 텍스트를 복사할 수 있어요.
B
복사한 텍스트를 아래 입력창에 붙여넣고 PDF로 변환하세요.
